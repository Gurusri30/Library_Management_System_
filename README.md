A simple Java Web Application to manage books and authors using:

Java Servlets

JDBC

Oracle/MySQL Database

Apache Tomcat

This project allows users to:

 Add a new Book

 Search/View a Book by ISBN

 Validate input data

 Store and retrieve data from database
